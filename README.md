# 前言

欢迎来到基于SSM的企业培训管理系统项目！该项目旨在帮助企业管理培训活动，提高员工技能和素质。本项目使用Java语言，结合Spring、SpringMVC和MyBatis框架，以及前端技术JS、Vue和CSS3进行开发。以下是项目的详细介绍。

## 内容介绍

本项目主要为企业提供一个便捷、高效的培训管理系统。系统主要包括以下功能模块：

1. 用户管理：包括管理员和普通员工账号的注册、登录、权限管理等功能。
2. 培训计划：管理员可以发布培训计划，员工可以查看并报名参加培训。
3. 培训资料：上传和分享培训相关资料，方便员工学习。
4. 培训评价：员工可以对培训课程进行评价，提高培训质量。
5. 数据统计：管理员可以查看培训参与情况、员工培训成绩等数据。

## 技术介绍

- 语言：Java
- 使用框架：Spring、SpringMVC、MyBatis
- 前端技术：JS、Vue、CSS3
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12/14/16
